# Back-end of online video game's shop

## Installation

```
$ npm install ||  $ npm i
```

## Run Application

```
 $ npm start
```

## Usage

`POST /register`

`POST /login`

`GET /auth/categories`

`GET /auth/items`

`POST /auth/cart:id`

`DELETE /auth/cart:id`

`GET /auth/cart`
